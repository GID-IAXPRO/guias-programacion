TEMA 7. Aspectos funcionales
1. ¿Qué es un puntero a una función? Pon un ejemplo de código en C, donde se define una función y que reciba una cadena de caracteres como parámetro y devuelva la cadena en mayúsculas. Crea un puntero en una variable local a dicha función llamado aMayusculas e invócala con el puntero.
Respuesta

Definición

Variable que apunta a una función.
Permite llamar funciones indirectamente.

Ejemplo C

#include <stdio.h>
#include <ctype.h>

void aMayusculas(char* s) {
    for (int i = 0; s[i]; i++) {
        s[i] = toupper(s[i]);
    }
}

int main() {
    void (*aMayusculasPtr)(char*) = aMayusculas;

    char txt[] = "hola";
    aMayusculasPtr(txt);
}
2. ¿Qué es una función lambda en un lenguaje de programación? Pon un ejemplo similar al anterior en Javascript y otro en Java con funciones lambda. Usa una variable local aMayusculas para apuntar a la función lambda.
Respuesta

Definición

Función anónima.
Definida en línea.

JavaScript

let aMayusculas = s => s.toUpperCase();

Java

Function<String,String> aMayusculas =
    s -> s.toUpperCase();

18. Ordenar lista de Persona por edad y nombre
Respuesta

Manual

Collections.sort(lista, (a,b) -> {
    if (a.edad == b.edad)
        return a.nombre.compareTo(b.nombre);
    return Integer.compare(a.edad,b.edad);
});

Comparator

Collections.sort(lista,
    Comparator.comparingInt(p -> p.edad)
              .thenComparing(p -> p.nombre)
);
